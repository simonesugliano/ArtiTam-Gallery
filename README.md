# ArtiTam-Gallery
Archivio pubblico per le immagini e il file gallery.json del sito dell’artigiano in cartongesso. Aggiornato automaticamente da Google Drive tramite Apps Script.


Repository dedicato allo storage statico delle immagini e del file gallery.json utilizzati dal sito React dell’artigiano in cartongesso.

Le foto vengono caricate dal cliente su Google Drive, uno script Apps Script genera il JSON e aggiorna automaticamente questo repository tramite API GitHub.

Contenuto:

/gallery/ → immagini suddivise per cantiere

gallery.json → indice completo usato dal front-end

Scopo: fornire una sorgente pubblica e stabile per la galleria del sito, senza limiti di CORS o accesso.
